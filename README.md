# Infrastructure Automation Blueprint

Этот репозиторий представляет собой учебный проект для автоматизации инфраструктуры уровня Senior.  
Он демонстрирует полный цикл: от первичного развёртывания виртуальных машин с помощью Terraform, настройки сервисов с помощью Ansible, 
развёртывания приложения и мониторинга через Docker, до документирования архитектуры и CI/CD.  
История коммитов показывает эволюцию проекта: сначала были проблемы с форматированием и правилами Ansible, а затем — постепенные улучшения, 
которые привели к успешному прохождению проверок Terraform и Ansible.

## Структура

```
.
├── ansible/               # Роли, плейбуки и инвентори Ansible
├── docker/                # Docker Compose и конфигурация сервисов
├── docs/                  # Диаграммы и документация
├── terraform/             # Конфигурация Terraform для Proxmox
├── .gitignore             # Исключения для Git
└── README.md              # Вы читаете его сейчас
```

## Как использовать

1. Клонируйте репозиторий и перейдите в директорию:
   ```bash
   git clone <URL>
   cd <repo-name>
   ```
2. Проверьте структуру проекта и изучите содержимое.  
3. Историю изменений можно просмотреть командой `git log --oneline --graph --all`.
